Lanyon. A Jekyll WebUI.

PURPOSE

This is a little project intended to help Jekyll users administrate their blogs without using FTP/SSH.

INSTALLATION

Edit config.php. You can generate a password using sha512sum.php (Running it from a shell WILL NOT work.)

NOTES

The project name comes from the book “Strange Case of Dr Jekyll and Mr Hyde”. Dr Hastie Lanyon was a friend of Dr Jekyll.
