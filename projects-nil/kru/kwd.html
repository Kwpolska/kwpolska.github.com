KWD. Awesome READMEs in 10 seconds flat.

PURPOSE

This project was made for speeding up making all the READMEs.

INSTALLATION

Check out INSTALL.md.

NOTES

There are two versions of KWD available: the Python version and the original Perl version. The Perl version is in original-perl/ directory. The Python version (KWD2/KWD v2) is the current version. You can convert your files using 1to2.py.

KWD stores config, licenses and the template in ~/.kwd/ (created by the config script).
